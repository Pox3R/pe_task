  • configureXMLAcceptRoute()

    Used to configure accept incomming POST on /convert requests with XML body. Response contains 200 - OK code and YAML-formatted data if XML is valid. Otherwise, it returns 400 - Bad Request code.

    Example:

    Request

    POST /convert

    Headers:

    Content-Type: application/xml

    Body:

    <person>
    <name>Eldar Goloviznin</name>
    <email>radleweird@gmail.com</email>
</person>

    Response

    Status: 200 - OK

    Body:

    person:
 email: radleweird@gmail.com
 name: Eldar Goloviznin
